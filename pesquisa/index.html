Middleware & Multer no Express.js

Aprenda sobre Middleware e Uploads de Arquivos no Express.js

O que é Middleware?

Middleware é uma função que tem acesso ao objeto de solicitação (req), resposta (res) e à próxima função no ciclo de requisição-resposta do Express.js.

Como funciona um Middleware?

Um middleware pode modificar a solicitação ou resposta antes de passar para o próximo middleware na pilha.

app.use((req, res, next) => {
    console.log('Middleware executado!');
    next();
});

Exemplos de Middleware no Express.js

  • Logger (registro de requisições)
  • Autenticação
  • Manipulação de erros

O que é Multer?

O Multer é um middleware do Express.js para lidar com uploads de arquivos.

Como configurar o Multer?

const multer = require('multer');
const upload = multer({ dest: 'uploads/' });
app.post('/upload', upload.single('arquivo'), (req, res) => {
    res.send('Arquivo enviado com sucesso!');
});
